# دليل التخصيص - Gold Market Admin Dashboard

## نظرة عامة
تم تخصيص لوحة التحكم لتتناسب مع مشروع سوق الذهب (Gold Market). تم تحديث الألوان والتصميم والعناصر لتتماشى مع طبيعة المشروع.

## التغييرات المنفذة

### 1. نظام الألوان (Color Scheme)
- تم تحديث الألوان الأساسية من الأزرق إلى الذهبي/البرتقالي
- الألوان الجديدة:
  - `brand-500`: `#ffa500` (ذهبي)
  - `brand-600`: `#e69400` (ذهبي داكن)
  - تم تحديث جميع درجات الألوان لتتناسب مع الطابع الذهبي

### 2. قائمة التنقل (Sidebar Navigation)
تم تحديث القائمة لتشمل العناصر التالية:

#### القائمة الرئيسية:
- **لوحة التحكم**: الرئيسية
- **المنتجات**: جميع المنتجات، إضافة منتج، الفئات
- **الطلبات**: جميع الطلبات، طلبات جديدة، طلبات قيد التنفيذ
- **العملاء**: قائمة العملاء، إضافة عميل
- **أسعار الذهب**: عرض الأسعار، تحديث الأسعار، التاريخ
- **التقارير**: تقارير المبيعات، تقارير المنتجات، تقارير العملاء

#### القائمة الإضافية:
- **الرسوم البيانية**: رسم بياني خطي، رسم بياني عمودي
- **الملف الشخصي**
- **الإعدادات**: الإعدادات العامة، الأمان

### 3. الهيدر (Header)
- تم تحديث نص البحث إلى: "ابحث في لوحة التحكم..."
- دعم اللغة العربية في الواجهة

### 4. الترجمة
- تم ترجمة عناصر القائمة إلى العربية
- تم تحديث النصوص لتتناسب مع مشروع سوق الذهب

## الملفات المعدلة

1. `src/styles.css` - تحديث نظام الألوان
2. `src/app/shared/layout/app-sidebar/app-sidebar.component.ts` - تحديث قائمة التنقل
3. `src/app/shared/layout/app-sidebar/app-sidebar.component.html` - تحديث النصوص
4. `src/app/shared/layout/app-header/app-header.component.html` - تحديث نص البحث

## الخطوات التالية (اقتراحات)

1. **إضافة شعار مخصص**: استبدال الشعار الحالي بشعار سوق الذهب
2. **تحديث الأيقونات**: استخدام أيقونات مناسبة للذهب والمعادن الثمينة
3. **إضافة صفحات جديدة**: إنشاء صفحات للمنتجات والطلبات والعملاء
4. **ربط APIs**: ربط لوحة التحكم بـ APIs المشروع
5. **إضافة دعم RTL**: تحسين دعم اللغة العربية واتجاه RTL

## كيفية التشغيل

```bash
cd gold-admin
npm install
npm start
```

ثم افتح المتصفح على: `http://localhost:4200`

## ملاحظات

- التصميم يدعم Dark Mode
